iOS开发者账号申请

发表于 2019-11-04 | 分类于 iOS |
字数统计: 1k

因公司业务需要，需要申请一个企业级的开发者账号。之前只申请过个人以及公司级的开发者账号，因此在申请企业级账号的同时顺便分享一下申请账号的流程。

Apple开发者账号类型

除了非营利、被认可的教育机构、政府组织之外，Apple开发者账号可以分为三类：

账号类型 申请对象 公司资质(邓白氏码) 使用者 发布App Store 适用范围 价格($/年)
个人开发者账号 个人开发者 不需要 个人开发者 可以 一般用于个人开发者使用 99
公司开发者账号 公司 需要 公司团队 可以 一般公司开发应用发布后供大家使用 99
企业开发者账号 公司 需要 企业团队 不可以 企业内部使用，一般只是用于公司内部测试或者内部使用 299

申请

  1. 申请Apple Id

    申请账号时就按照提示填写完相关信息即可。记住你的安全问题的答案，更改时需要使用

    注意⚠️：申请完Apple Id后你需要登陆你的账号去补充联系方式和开启双重认证，否则在注册开发者账号时会提示你去补充相关信息，如下图显示

  2. 申请开发者账号

    个人以及公司开发者申请

    企业开发者申请

    1. 登陆你的账号，然后进入相应链接，首先你需要同意相关条例。

    2. 选择加入开发者类型

      个人以及公司入口

      企业开发入口

    3. 如果选择的是个人开发者只需要按照提示填写完相关信息即可，最后完成付款等待消息。

    4. 如果是公司或者企业级开发者账号就需要公司资质认证，确保公司符合以下条件。

      • 合法的公司主体
      • 邓白氏码

      • 企业网址(已通过备案的合法网址)

        注意⚠️：邓氏编码(D-U-N-S® Number)是一种实时动态的企业身份标识。它源自于一个独一无二的9位数字全球编码系统DUNS，相当于企业的身份识别码(就像是个人的身份证)，被广泛应用于企业识别、商业信息的组织及整理。可以帮助识别和迅速定位全球2.4亿家企业的信息,这个是申请公司级或者企业级开发者账号必须的。

        邓白氏码并不一定要通过Apple的官网提示去申请，你也可以到邓白氏码官网去申请，只有确保拥有邓白氏码即可。

        如果你你不确定公司是否已经申请，可以使用Apple提供的查找工具，即点击Check Now

    5. 申请邓白氏码

      如果公司还未申请过邓白氏码，上一步提交验证码之后，会提示当前未申请邓白氏码，将企业信息如实填写完成邓白氏码的申请。

      注意⚠️：申请邓白氏码后需要注意查看邮箱，因为之后会收到邓白氏的邮件，你需要按照邮件的提示去补充完善相关信息。

    6. 完成邓白氏码的申请后，我们按照申请提示继续完成开发者账号的申请。

      选择申请人类型

      • 公司负责人：公司的创始人，选中这个，不需要填写个人信息。

      • 公司委托人：建议选择这个，选中后补全个人信息，否则后期可能会因为身份信息带来一些不必要的麻烦

        按照实际将信息填写完成，并提交后，会提示你已经进行Apple开发者账号申请，下面耐心等待苹果的回馈邮件即可。

注意⚠️：公司级开发者账号申请周期一般在一周左右。企业级开发者账号现在审核非常严格，申请周期也很长，而且通过率比较低。
